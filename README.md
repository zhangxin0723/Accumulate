#1.项目简介
积纳有品--微信小程序

#2.技术栈
主框架为小程序，结合 Taro,mobx 等运行。

#3.项目运行

```
## 初始化项目 
npm run dev:weapp

## 安装依赖
npm i   

## 指定平台的开发时构建(微信、百度、头条、支付宝)
npm dev:wx
npm dev:swan
npm dev:tt
npm dev:my

## 指定平台的打包构建
npm build:wx
npm build:swan
npm build:tt
npm build:my
